# Proposal Website for Aradhya

A beautifully designed proposal website with soft animations, heartfelt messages, and an elegant UI.

Made with love by Jeet.

---

## Tech Stack

- Next.js
- Tailwind CSS
- Motion (animations)
- Swiper.js (image carousel)

---

## Running Locally

```bash
npm install
npm run dev
```

Open http://localhost:5000 in your browser.
